## Install / Setup

**MacOS**
```bash
brew install jakewmeyer/Geo/geo
```

## Usage

### Overwrite existing .gitignore file
```bash
zeus > .gitignore
```

### Append existing .gitignore file
```bash
zeus >> .gitignore
```
